# 抖音极速版 GKD 自动滑屏指南

## 文件
- 规则：`gkd_douyin_swipe.json5`
- 截图/快照：用户微信文件 `抖音极速版_MainActivity-*` 文件夹

## 导入步骤
1. 把 `gkd_douyin_swipe.json5` 放到手机里任意目录（如 `/sdcard/Download/`）
2. GKD → 订阅 → 本地订阅 → 右上角"+" → 选择该文件
3. 找到「**自动滑屏**」规则组 → 开启

## 匹配策略（v3 修正版，2026-06-25）

### 目标节点
```
ViewPager[vid="viewpager"][desc="视频"][visibleToUser=true]
```
- 节点 id=42，bounds=(0, 0, 1260, 2650)
- `visibleToUser=true`，覆盖整个视频播放区
- 赚钱任务页也有此节点 → **必须配合 excludeMatches**

### 排除赚钱任务页
```
excludeMatches:
  - ViewGroup[desc*="赚金币"]  // 赚钱页 desc='新人看视频赚金币'
  - ViewGroup[desc*="签到"]    // 赚钱页 desc='签到宝箱全不忘...' / desc='距7月14日签到结束...'
```

### 为什么不用 TextView[vid="desc"]？
抖音主页的 `vid="desc"` 节点是视频描述：
- `visibleToUser=false`（不在屏幕可见区域内）
- 坐标越界（B=-39 / B=2650）
- **GKD 不会匹配可见性 false 的节点** → 永远不滑屏

### 排除赚钱任务页
赚钱任务页**也是 MainActivity**（不是独立 Activity），也有 ViewPager[vid="viewpager"]！
所以必须用 `excludeMatches` 排除赚钱页独有的标识节点：

```
excludeMatches:
  - ViewGroup[desc*="赚金币"]  // 赚钱页 desc='新人看视频赚金币'
  - ViewGroup[desc*="签到"]    // 赚钱页 desc='签到宝箱全不忘...' / '距7月14日签到结束...'
```

这些描述**仅在赚钱任务页出现**，视频主屏完全不存在。

### 为什么不用 TextView[text="开宝箱"] 排除？
- 抖音主页底部**有金币悬浮按钮** `text="开宝箱"`，bounds=(663, 2619, 773, 2662)
- 这个按钮在主页和"赚钱任务页"**都存在**，用它排除会**误伤主页**
- → 改用赚钱页独有的 `desc` 属性排除

## 滑屏规则
| 规则 | 间隔 | 起始点 | 结束点 | 持续 |
|------|------|--------|--------|------|
| 滑屏A | 5秒 | 中下 | 中上 | 380ms |
| 滑屏B | 7秒 | 左下 | 左上 | 420ms |
| 滑屏C | 9秒 | 右下 | 中上 | 310ms |
| 滑屏D | 11秒 | 中下 | 中上 | 360ms |
| 滑屏E | 13秒 | 中下 | 中上 | 450ms |
| 备用-评论 | 6秒 | 中下 | 中上 | 400ms |

5 条规则间隔错开（5/7/9/11/13 秒），GKD 会按 `key` 顺序串行触发，
形成"5→7→9→11→13→5→..."的循环，更像真人随机行为。

## 验证方法
1. 打开抖音极速版，停留在「推荐」tab 视频界面
2. 等待 5-13 秒，应看到视频自动上滑
3. 切到赚钱任务界面（底部金币按钮 → 赚钱），应**停止滑动**
4. 切回视频主屏，应恢复滑动
5. 切到「我」「朋友」等页，应停止滑动
6. 切到「直播」「商城」频道 tab（仍在 MainActivity 内）也会滑动
   - 如果不想在直播 tab 滑：加 excludeMatches: `TextView[desc*="直播间"]`

## 常见问题

### Q: 规则开了但完全不滑动？
A: 90% 是 `matches` 选择器不命中。用 GKD 自带的"测试规则"功能，
   或对照快照确认 `ViewPager[vid="viewpager"]` 节点存在。

### Q: 滑得太快/太慢？
A: 调 `actionCd`（下次滑动间隔）和 `actionDelay`（触发后等待）。
   `actionCd` 建议保持在 4000ms 以上，模拟真人。

### Q: 直播/带货页也滑屏？
A: 因为直播间内也有 video viewpager。
   解决：加 `excludeMatches: ["View[desc*=\"点击进入直播间\"]"]`

### Q: 滑屏被抖音检测？
A: 滑太快（<300ms）、太规律（同间隔）、太精准（落点重合）都易被检测。
   建议：`duration: 300-500ms` 随机 + 起点横坐标 0.35-0.55 随机。
