# GKD 抖音极速版自动滑屏 - 使用指南

## 📋 前置条件

1. 安装 GKD 应用：https://github.com/gkd-kit/gkd/releases
2. 授予 GKD **无障碍权限**
3. 手机上安装抖音极速版（包名：`com.ss.android.ugc.aweme.lite`）

---

## 🚀 导入规则

### 方法：手动输入

1. GKD → 订阅 → 添加应用规则
2. 将文件里的内容复制进去
3. 打开本地订阅里的应用规则

---

## ⚙️ 规则说明

| 规则组 | 默认状态 | 说明 |
|--------|---------|------|
| **自动滑屏** | ❌ 关闭 | 模拟真人5-13秒不规律间隔自动上滑 |

手动开启方式：GKD → 订阅 → 本地订阅 → 抖音极速版 → 自动滑屏 → 打开开关

---

## 🎲 模拟真人随机原理

GKD不支持随机间隔，所以用了5条规则同时运行，各自独立计时：

| 规则 | 间隔 | 滑动起点 | 滑动速度 |
|------|------|---------|---------|
| A | 5秒 | 偏中间 | 380ms |
| B | 7秒 | 偏左 | 420ms |
| C | 9秒 | 偏右 | 310ms |
| D | 11秒 | 正中间 | 360ms |
| E | 13秒 | 偏左下 | 450ms |

谁先到时间就谁触发，实际效果就是滑屏间隔在 **5~13秒之间不规律跳动**，看起来像真人有时候刷得快、有时候看得久。每条规则的滑动起点和速度也不同，进一步模拟真人。

---

## 🔧 修改滑屏间隔

规则文件里每个规则的 `actionCd` 控制间隔（单位毫秒），两个规则都改一样的值即可：

| 想要的效果 | actionCd 值 |
|-----------|------------|
| 快速刷 | `3000`（3秒） |
| 中速 | `5000`（5秒） |
| 慢速 | `8000`（8秒） |

---

## 🔧 选择器匹配原理

规则基于手机快照分析，使用稳定特征识别视频播放页：

| vid | 节点 | 用途 |
|-----|------|------|
| `desc` | 视频描述文本 | 主选择器：识别播放页 |
| `comment_layout` | 评论按钮 | 备用识别方式 |

## ⚠️ 防检测建议

1. **偶尔手动操作** — 点个赞、暂停一下、评论一下
2. **不要连续跑太久** — 刷1-2小时就休息
3. **不要全靠自动** — 真人不可能只滑不看，偶尔让它停在某个视频久一点

---

## ⚠️ 注意事项

1. **vid 会随版本变化**：抖音极速版更新后，vid（如 `desc`）可能改名，失效时需重新抓快照更新选择器
2. **forcedTime = 15秒**：视频播放时界面几乎不变，GKD收不到节点事件，所以强制每15秒查一次
3. **手动开关**：规则默认关闭，想刷就开、不想刷就关
